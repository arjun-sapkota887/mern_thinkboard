using mern tech stack to create a note taking web app

M- mongoDB: NoSQL database to store the data
E - Express: web framework, we use framework so we don't have to write the code from scratch
    A web framework is a ready to use toolbox for building web apps faster and more easily
R - React: for frontend library
N - Node JS: javascript runtime: allows us to run JS in server insted of on client

CRUD routes - four basic database operations (Create, Read, Update, Delete)
nodemon (a tool that auto-restarts the server when files change)
